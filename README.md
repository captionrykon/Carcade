## Getting Started

1. Clone repo

```bash
git clone https://github.com/Siddesh7/vibe-coding-frames-starter.git
```

2. Install dependencies

```bash
pnpm install
```

3. Environment setup  
   Create `.env.local` with:

```env
NEXT_PUBLIC_PRIVY_APP_ID=
NEXT_PUBLIC_URL=
```

4. Run dev server

```bash
pnpm run dev
```
